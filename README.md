# LifeOS

Kişisel yönetim uygulamam. Görevler, notlar, finansal takip ve daha fazlası için geliştirdiğim modern bir web uygulaması.

## Hakkında

Bu proje, günlük yaşamımı organize etmek ve finansal durumumu takip etmek için geliştirdiğim kişisel bir uygulamadır. Supabase ile backend entegrasyonu, modern React mimarisi ve kullanıcı dostu arayüz ile geliştirilmiştir.

## Teknik Detaylar

- **Frontend**: React + TypeScript + Vite
- **UI Framework**: Tailwind CSS + shadcn/ui
- **Animasyonlar**: Framer Motion
- **State Management**: Zustand
- **Backend**: Supabase (PostgreSQL + Storage + Auth)
- **Grafikler**: Recharts
- **Tarih İşlemleri**: date-fns

## Not

Bu proje kişisel kullanım için geliştirilmiştir. Kod yapısı ve mimari, benim ihtiyaçlarıma göre özelleştirilmiştir.
